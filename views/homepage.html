EZMessage

A new way to message

About The App and More...

An application that will allow a user to be able to log in and out, view when other users are online or offline, and send and recieve messages that will show a time stamp of when it was sent. Messages that a user will be able to send will include emojis. A user will be able to create set times to send premade messages and a user will be able to delete previously sent messages.
